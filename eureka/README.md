# Eureka

## 概念
```
Eureka注册中心(服务治理)是 Spring Cloud Eureka 的简称,是 Spring Cloud Netflix 组件之一。
```

## 特性
>Eureka既可做为服务端,也可以做为客户端

### 服务端
    – 管理服务实例
    – 提供服务注册或下线
    – 提供服务发现
    – 提供服务注册表至两类客户端
### 客户端
    – 连接服务注册中心
    – 向服务注册中心注册或者下线服务实例
    – 向服务注册中心或服务注册缓存列表查询服务
